# Valentine's Day Proposal Site ❤️

A simple, interactive Valentine's Day proposal site from Amar to Radhika.

## Features
- Interactive "Yes" and "No" buttons.
- The "No" button runs away when hovered!
- Confetti animation upon clicking "Yes".
- Responsive design.

## How to Host on GitHub Pages
1. Create a new repository on GitHub named `amar-rad-valentines`.
2. Upload `index.html`, `style.css`, and `script.js` to the repository.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, set the source to **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then click **Save**.
6. Your site will be live at `https://<your-username>.github.io/amar-rad-valentines/`.
